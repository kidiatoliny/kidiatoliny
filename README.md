`Laravel` · `React` · `React Native` · `PostgreSQL` · `Redis`

[Explore NosFerry](https://nosferry.com)

## ~/open-source

<details>
<summary><strong>Payments and fiscal systems</strong> · gateways, billing, and SDKs</summary>

- [Laravel SISP](https://github.com/akira-io/laravel-sisp) and [Node SISP](https://github.com/akira-io/node-sisp) integrate the SISP and Vinti4 payment gateway for Cabo Verde.
- [Payable](https://github.com/akira-io/payable) is a provider-independent billing engine for Node.js.
- [Node e-Fatura](https://github.com/akira-io/node-efatura) validates Cabo Verde fiscal documents.
- Akira Billing SDKs share the billing protocol across [Go](https://github.com/akira-io/billing-sdk-go), [Rust](https://github.com/akira-io/billing-sdk-rust), and [TypeScript](https://github.com/akira-io/billing-sdk-js).

</details>

<details>
<summary><strong>Laravel foundations</strong> · reusable application building blocks</summary>

- [QR Code](https://github.com/akira-io/laravel-qrcode) and [PDF Invoices](https://github.com/akira-io/laravel-pdf-invoices) cover document generation.
- [Auth Logs](https://github.com/akira-io/laravel-auth-logs), [Commentable](https://github.com/akira-io/laravel-commentable), and [Audit Chain](https://github.com/kidiatoliny/laravel-audit-chain) add activity, collaboration, and immutable audit trails.
- [License Core](https://github.com/kidiatoliny/laravel-license-core) provides licensing domain logic without imposing a UI or delivery layer.
- [Spectra](https://github.com/kidiatoliny/laravel-spectra) turns Laravel routes into an interactive API inspector.

</details>

<details>
<summary><strong>Systems and developer tooling</strong> · cross-platform foundations</summary>

- [Onyx](https://github.com/akira-io/onyx) and [Onyx RS](https://github.com/akira-io/onyx-rs) provide cross-platform desktop foundations in Go and Rust.
- [Git Cognition](https://github.com/akira-io/git-cognition-rs) abstracts version-control providers; [Omnitrack](https://github.com/akira-io/omnitrack-rs) does the same for issue trackers.
- [Graphify](https://github.com/kidiatoliny/graphify) turns code, schemas, documents, and media into a queryable knowledge graph.
- [ElasticLens](https://github.com/kidiatoliny/elasticlens) brings Elasticsearch search to SQL-backed Laravel models.

</details>

<details>
<summary><strong>Applied intelligence</strong> · retrieval and structured data</summary>

- [Laravel RAG](https://github.com/kidiatoliny/laravel-rag) provides retrieval-augmented generation with PostgreSQL, pgvector, and Prism.
- [Data Wizard](https://github.com/kidiatoliny/data-wizard) extracts structured data from PDFs, documents, and images.
- [Extractor](https://github.com/kidiatoliny/extractor) brings AI-assisted data extraction into Laravel applications.

</details>
